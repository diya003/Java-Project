# AirlineApp
## CLI Airline Management System 

AirlineApp is a robust, feature-rich Command Line Airline Management System built entirely using core Java.  
It simulates a real-world airline booking experience directly inside the terminal using ANSI escape codes, CSV-based persistence, and role-based access control.

## Overview

AirlineApp provides a graphical-like terminal interface for airline management without using any external frameworks or databases.

The system handles the complete lifecycle of an airline reservation platform:
- User authentication and registration
- Flight search and booking
- Seat allocation and fare calculation
- Ticket generation and cancellation
- Administrative analytics and management





---
## Getting Started

Follow these steps to get the application running on local machine.

---
## Step 1: Prerequisites

Ensure you have the following installed:

- **Java Development Kit (JDK):** Version 8 or higher
- **Terminal:** A terminal that supports ANSI colors (VS Code Terminal, IntelliJ IDEA, Linux Terminal, PowerShell, or Git Bash)

---

## Step 2: Compile the Code

Open your terminal in the directory where `AirlineApp.java` is located and run:

```bash
javac AirlineApp.java
```



Once **compilation is successful**, start the application using:

```bash
java AirlineApp

```

## System Architecture
User (CLI)
↓
Controller Layer
↓

- Singleton pattern for centralized data storage
- Factory-style object creation methods
- MVC-like separation of concerns
